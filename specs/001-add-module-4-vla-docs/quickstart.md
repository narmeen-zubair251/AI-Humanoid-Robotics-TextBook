# Quickstart: Module 4 VLA Documentation

This quickstart guide provides instructions on how to set up and view the generated Module 4 Vision-Language-Action (VLA) documentation within the Docusaurus project.

## 1. Verify File Structure

Ensure the following files and directories exist in your project:

```
my-book/
└── docs/
    └── module-04-vla/
        ├── index.md
        ├── lesson-01-voice-to-action.md
        ├── lesson-02-cognitive-planning.md
        └── lesson-03-autonomous-humanoid.md
```

## 2. Install Dependencies (if necessary)

Navigate to the `my-book` directory and install the Docusaurus dependencies if you haven't already:

```bash
cd my-book
npm install
```

## 3. Start the Docusaurus Development Server

From the `my-book` directory, start the development server:

```bash
npm run start
```

This will open a new browser window (usually at `http://localhost:3000`) displaying your Docusaurus site.

## 4. Navigate to Module 4

In the Docusaurus site, locate "Module 4: Vision-Language-Action (VLA)" in the sidebar navigation to view the newly added documentation.
