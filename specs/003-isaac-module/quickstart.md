# Quickstart: Integrating Module 3 into Docusaurus Sidebar

**Feature**: `003-isaac-module`
**Date**: 2025-12-12
**Spec**: specs/003-isaac-module/spec.md

## Overview

This quickstart guide provides instructions on how to integrate the newly created Module 3, "The AI-Robot Brain (NVIDIA Isaac™)", into your existing Docusaurus documentation sidebar. After generating the markdown files for Module 3, you will need to update the `sidebars.ts` file in your Docusaurus project to make the module discoverable and navigable.

## Prerequisites

-   Your Docusaurus project is set up and running.
-   The markdown files for Module 3 are located in `my-book/docs/module-03-isaac/`.

## Steps to Integrate

### 1. Locate `sidebars.ts`

Navigate to the `my-book` directory of your Docusaurus project and locate the `sidebars.ts` file. This file is typically found at `my-book/sidebars.ts`.

### 2. Edit `sidebars.ts`

Open `my-book/sidebars.ts` in your preferred code editor. You will need to add a new sidebar entry for `module-03-isaac`.

Find the existing `docsSidebar` array (or similar array that defines your documentation categories). Add a new object to this array, ensuring it follows the existing structure.

**Example `sidebars.ts` before (snippet):**

```typescript
import type { SidebarsConfig } from '@docusaurus/plugin-content-docs';

const sidebars: SidebarsConfig = {
  // By default, Docusaurus generates a sidebar from the docs folder structure
  tutorialSidebar: [{ type: 'autogenerated', dirName: '.' }],

  tutorialSidebar: [
    'intro',
    {
      type: 'category',
      label: 'Tutorial - Basics',
      items: ['tutorial-basics/create-a-document', 'tutorial-basics/create-a-blog-post', 'tutorial-basics/create-a-page', 'tutorial-basics/deploy-your-site', 'tutorial-basics/markdown-features'],
    },
    {
      type: 'category',
      label: 'Module 01 - ROS 2',
      items: [
        {
          type: 'autogenerated',
          dirName: 'module-01-ros2',
        },
      ],
    },
    {
      type: 'category',
      label: 'Module 02 - Digital Twin',
      items: [
        {
          type: 'autogenerated',
          dirName: 'module-02-digital-twin',
        },
      ],
    },
  ],
};

export default sidebars;
```

**Example `sidebars.ts` after adding Module 3 (snippet):**

```typescript
import type { SidebarsConfig } from '@docusaurus/plugin-content-docs';

const sidebars: SidebarsConfig = {
  // By default, Docusaurus generates a sidebar from the docs folder structure
  tutorialSidebar: [{ type: 'autogenerated', dirName: '.' }],

  tutorialSidebar: [
    'intro',
    {
      type: 'category',
      label: 'Tutorial - Basics',
      items: ['tutorial-basics/create-a-document', 'tutorial-basics/create-a-blog-post', 'tutorial-basics/create-a-page', 'tutorial-basics/deploy-your-site', 'tutorial-basics/markdown-features'],
    },
    {
      type: 'category',
      label: 'Module 01 - ROS 2',
      items: [
        {
          type: 'autogenerated',
          dirName: 'module-01-ros2',
        },
      ],
    },
    {
      type: 'category',
      label: 'Module 02 - Digital Twin',
      items: [
        {
          type: 'autogenerated',
          dirName: 'module-02-digital-twin',
        },
      ],
    },
    // START: New Module 3 Entry
    {
      type: 'category',
      label: 'Module 03 - AI-Robot Brain (NVIDIA Isaac™)', // Display label for the module
      items: [
        {
          type: 'autogenerated',
          dirName: 'module-03-isaac', // This should match the folder name in my-book/docs
        },
      ],
    },
    // END: New Module 3 Entry
  ],
};

export default sidebars;
```

**Explanation of changes:**

-   A new object `{ type: 'category', ... }` is added to the `tutorialSidebar` array.
-   `label`: This is the text that will appear in the sidebar for the module (e.g., "Module 03 - AI-Robot Brain (NVIDIA Isaac™)").
-   `items`: Contains an `autogenerated` object.
    -   `dirName`: This crucial property tells Docusaurus to automatically generate sidebar links for all markdown files found within the `my-book/docs/module-03-isaac` directory.

### 3. Verify Integration

After saving the `sidebars.ts` file, run your Docusaurus development server:

```bash
cd my-book
npm run start
```

Open your browser and navigate to your Docusaurus site. You should now see "Module 03 - AI-Robot Brain (NVIDIA Isaac™)" in the sidebar, with its chapters listed underneath. Click on each chapter to ensure they render correctly.
